Airport API

Version: 1.0.0

Airport statistics API design documentation.

Schemes: https, http

Summary

Tag: airports

Operation Description
GET /airports

Returns all airports in the US.
GET /airports/{airportId}/carriers

Returns all carriers operating at a specific US airport.

Tag: carriers

Operation Description
GET /carriers

Returns all carriers operating in US airports
GET /carriers/delays

Returns a list of per-carrier delays

Tag: statistics

Operation Description
GET /airports/{airportId}/carriers/{carrierId}/statistics

All statistics about flights of a carrier to/from a US airport for a given month/all months available.

POST /airports/{airportId}/carriers/{carrierId}/statistics

All statistics about flights of a carrier to/from a US airport for a given month/all months available.

PUT /airports/{airportId}/carriers/{carrierId}/statistics

Update statistics about flights of a carrier to/from a US airport for a given month/all months available.

DELETE /airports/{airportId}/carriers/{carrierId}/statistics

Delete statistics for a given combination of Airport, Carrier and Month+Year.
GET /airports/{airportId}/carriers/{carrierId}/flights

Returns the number of on-time, delayed, and cancelled flights of a carrier.
GET /carriers/{carrierId}/delays/statistics

Get mean, median, standard deviation for a specific carrier and route.
GET /carriers/delays/statistics

Get mean, median, standard deviation for all carriers and a given route.
